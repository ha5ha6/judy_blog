### Python String Operation

1. check if char is in alphabet  
s='2'  
s.isalpha()  
False  
s='aBcD'  
s.isalpha()  
True

2. remove white spaces before and after  
s='  lajdflak    '  
s=s.strip()  
s='lajdflak'

3. swtich to upper case and lower case  
s='aBcDeFg'  
s=s.lower()  
s='abcdefg'  
s=s.upper()  
s='ABCDEFG'
